						Додаток "Чат"

Режим тесту на одному компьюетрі:

 - Запустити в терміналі python3 Serv.py
 - Запустити в іншому терміналі Clie.py
 - Запустити ще один терміналі Clie.py

Режим локальної мережі:

 1. Запустити файл Serv.py на компютері: python3 Serv.py

 2. Запустити файл Clie.py на тому ж компьютері окремому терміналі: python3 Clie.py (для можливості переписки, якщо ні - пропустити цей пункт)

 3. Запустити файл Clie.py на машині в одній локальній мережі з Serv.py
   Для запуску використовувати: python3 Clie.py <ip> <port> (наприклад, python3 Clie.py 192.0.0.1 5001).

 4. Запустити файл Clie.py на інших машинах анологічно пунтку 3

Особливості:
 - Можна підключати більше 2 машин.
 - Після того, як клієнти покинули чат і він пустий, пропонується його відключити з періодчністю в декілька секунд.
 - Не повинні виникати помилки, якщо клієнт покинув чат не за домопогою команди exit.
 - Клієнт може повторно підключатись після виходу з чату через новий коннект, сервер без команди не прериваю роботу і не закриває свій сокет.
 - Якщо сервер припинив роботу у клієнта не виникає помилки, сокет закривається.
 - IP та порт при запуску фалу Clie.py можна вводити в консолі, а можна не вводит, вона зашита в программі (для тестового режиму).
